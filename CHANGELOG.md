# 変更履歴

このプロジェクトの主な変更はこのファイルに記録します。形式は Keep a Changelog に基づきます。

## [Unreleased]

### Added

- 非公式APIから作品一覧、検索、作品詳細、章画像を安全に取得して閲覧できる、n-mgram の初期デスクトップ版を追加。
- 大量画像の同時読み込みを抑える縦読み、キーボード操作に対応したページ読み、表示サイズ切り替えを追加。
- 途中から読書を再開できる読書位置と、お気に入りを端末内へ保存する機能を追加。
- API仕様変更、タイムアウト、アクセス制限を区別し、復旧方法を判断できるエラー表示を追加。
- 読んだ作品名、章、章内の進捗率、最終閲覧日時を一覧でき、作品単位で削除できる読書履歴を追加。
- 読書履歴の作品について最新章を確認し、未読の新しい章をまとめて開ける通知センターを追加。
- 作品APIで利用できる全メタデータを対象に、文字列、数値範囲、日付範囲、連載・表示状態を組み合わせて絞り込み、任意項目で昇順・降順に並べ替えられる詳細検索を追加。
- ディスカバー、検索、ヒストリー、アップデートを独立ページ化し、履歴の一括削除を追加。
- 署名済みGitHub Releaseを確認し、アプリ内で更新・再起動できるTauri updaterを追加。
- GitHub Actionsによる品質検査、Windowsコンパイル、タグ起点の署名付きRelease生成を追加。
- Android向けTauriプロジェクト、署名付きAPKの直接配布、アプリ内の新バージョン確認とブラウザ経由の更新導線を追加。
- 小画面では4ページを下部ナビゲーションへ配置し、縦持ちは1ページ、横持ちは右綴じ見開きで表示するモバイルレイアウトを追加。
- タッチ操作向けに、左タップ・左スワイプで次へ、右タップ・右スワイプで前へ、中央タップで操作UIを切り替えるページ操作を追加。

### Changed

- API側の互換判定に備え、TauriのネイティブHTTP通信から解析元アプリと同じUser-Agent、SDKバージョン、アプリバージョンヘッダーを送るよう変更。
- 日本漫画を本として自然な方向で読めるよう、ページ読みを表紙右配置の見開き表示にし、若いページを右、次ページを左へ置く右綴じ操作へ変更。
- 閲覧へ集中できるよう、操作が止まるとビューアのヘッダーと操作ヒントを自動的に隠し、ポインター・キーボード操作で再表示する挙動へ変更。
- 漫画や章を開き直すたびに表示を選び直さなくて済むよう、ビューアの縦読み・ページ読みと表示サイズの選択を端末内へ保存するよう変更。
- アプリ内の文言を日本語・英語の両方で利用できるよう整備。
- 既存ユーザーの読書位置とお気に入りを保ったまま履歴情報を追加できるよう、ローカルライブラリをバージョン2へ自動移行するよう変更。
- 非公式APIの仕様変更時に修正箇所をすぐ把握できるよう、接続先、偽装ヘッダー、レスポンス検証、データ正規化、画像処理を単一のAPI実装ファイルへ集約。

### Fixed

- Newest一覧に更新日時が未設定の作品が含まれても一覧全体が取得失敗にならないよう、欠損日時のレスポンス検証を修正。
- Popularの長期連載作品に空の画像URLが混ざっても作品全体が取得失敗にならないよう、欠損ページだけを除外する章データ検証へ修正。
- 作品説明に含まれるHTMLタグや文字参照がそのまま画面へ出ないよう、安全なプレーンテキストへの正規化を追加。
- 翻訳者募集の案内画像が漫画ページへ混入しないよう、既知URLに加えて画像内容のSHA-256ハッシュを表示前に検証する除外処理を追加。
- ジャンル検索をAPI必須形式のオブジェクト配列で送信し、`psychological`などの検索がHTTP 400になる問題を修正。
